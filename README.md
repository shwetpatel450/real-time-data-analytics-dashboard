# 📈 Real-Time Data Analytics Dashboard

## 📌 Project Overview
This project is a real-time dashboard that fetches live data from public APIs and displays continuously updated insights using Streamlit. It demonstrates how real-world monitoring systems process and visualize live information.

## 🎯 Objectives
- Fetch live data using APIs.
- Process and clean incoming data.
- Display real-time insights.
- Auto-refresh dashboard.
- Provide interactive filtering.

## 🛠️ Technologies Used
- Python
- Streamlit
- Requests
- Pandas

## 📂 Data Source
- OpenWeatherMap API
or
- CoinGecko API

## 📊 Dashboard Features
- Live API Data
- Auto Refresh
- Interactive Dashboard
- Dynamic Metrics
- Real-Time Updates
- User-Friendly Interface
